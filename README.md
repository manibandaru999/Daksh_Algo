# Daksh Algo Solutions Website

Static website for Daksh Algo Solutions showcasing services for algo trading, trade suggestions and custom applications.

## Pages
- `index.html` – Home page with overview and plan highlights.
- `about.html` – Company vision and why choose us.
- `services.html` – Details of offered services.
- `pricing.html` – Subscription plan information.
- `contact.html` – Contact form and contact details.

## Usage
Open `index.html` in a web browser to view the site.
